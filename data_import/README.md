## 🔧 インポート方法

### 管理画面からのインポート

1. アプリケーションにログイン
2. 管理ダッシュボード → CSV管理 にアクセス
3. 以下の順序でインポート:
   1. **shooting_categories.csv** を先にインポート
   2. **product_categories.csv** を次にインポート
   3. **items.csv** を最後にインポート（product_category_idの参照が必要なため）

### 注意事項

- **shop_id**: すべて `1` に設定されています。別のshop_idを使用する場合は一括置換してください
- **id**: 連番が振られています。既存データとの重複を避けたい場合は削除または調整してください
- **インポート順序**: 外部キー制約があるため、上記の順序を守ってください
